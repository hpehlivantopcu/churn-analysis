## How to Swap In Real Data
The script currently uses `generate_telecom_data()` which creates a synthetic dataset.

To use a **real CSV**, find this line near the top:
```python
df = generate_telecom_data(n=5000)
```
Replace it with:
```python
df = pd.read_csv("your_actual_data.csv")
```
Then make sure your CSV column names match or update the column references in the EDA and modeling sections.
